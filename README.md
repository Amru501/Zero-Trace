# 🕵️ Zero Trace

> Coordinate a prison break through a calculator.

Zero Trace is a story-driven hacking puzzle game where you act as the field operative. Ghost, an AI hacker, communicates with you through a secure chat while Kai, trapped inside the facility, can only receive numbers through a calculator display.

Every mission is dynamically generated with Ollama, creating unique scenarios, puzzles, and objectives.

## ✨ Features

- 🤖 AI-generated missions powered by Ollama
- 💬 Real-time hacker chat interface
- 🧩 Puzzle-driven gameplay
- 🔢 Calculator-based communication system
- 🚨 Alert and glitch mechanics
- 🔄 Replayable missions with unique scenarios
- 🎮 Immersive terminal-inspired UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Ollama

## 🚀 Getting Started

### Prerequisites

- Node.js
- Ollama running locally

### Installation

```bash
git clone https://github.com/Amru501/Zero-Trace.git
cd Zero-Trace
npm install
npm start
```

Then open:

```text
http://localhost:3000
```

## 🎯 Gameplay

- Receive instructions from **Ghost**
- Solve puzzles and decode information
- Use the calculator to transmit numbers to **Kai**
- Avoid raising the alert level
- Complete the mission and escape without leaving a trace
